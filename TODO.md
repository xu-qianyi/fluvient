## 1. 产品方向 / 待决策

- [ ] **评估 Chrome 插件形态**
  竞品已验证这条路走得通:
  - **InsightTube** —— 插件在 YouTube 视频**旁边**展开侧边栏(Summary / Chapters / Chat),不碰视频本身。
  - **Fluent AI** —— 插件直接在 YouTube 视频**画面上方**叠加双语字幕(英文 + 中文逐词对照),带自己的播放控制条。

  结论:Chrome 插件可通过注入 DOM / content script,直接在 YouTube 原生播放器上覆盖字幕层,**不需要**把视频搬到自建页面播放。
  待定:Fluvient 是否要从现在的自建播放页(`/watch/[videoId]`)转向/新增 Chrome 插件这条分发路径?

- [ ] **多语言方向落地**
  产品目标已从「中文用户学英语」扩展为**多语言**(2026-06-16 确定)。
  现有 v1 是按中文→英语搭建(中文 UI、笔记中文释义),i18n 框架已就位。
  待定:支持哪些学习语言 / 母语组合?释义与笔记的多语言生成策略?
